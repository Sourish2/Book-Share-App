# Book Share App

## Overview

Book Share App is a web-based EPUB library and reader platform built with React and Express. Users can upload EPUB books and optional cover images, browse featured books through a 3D carousel, read books online using an EPUB reader, and download books directly from cloud storage.

The project is currently transitioning from local filesystem storage to AWS S3.

        ▼

## S3 Storage Layout

```text
bucket-name/

├── books/
│   ├── Dune.epub
│   ├── Foundation.epub
│   └── Hyperion.epub
│
└── covers/
    ├── Dune.jpg
    ├── Foundation.jpg
    └── Hyperion.jpg
```

Naming convention:

```text
Book:
books/<Title>.epub

Cover:
covers/<Title>.jpg
```

Example:

```text
books/Dune.epub
covers/Dune.jpg
```

## Backend Endpoints

### Upload Book

```http
POST /api/upload
```

Multipart form fields:

```text
book    (required)
cover   (optional)
```

Uploads files to S3.

---

### Read Book

```http
GET /api/books/:filename
```

Streams an EPUB directly from S3.

Example:

```http
GET /api/books/Dune.epub
```

---

### Download Book

```http
GET /api/books/download/:filename
```

Downloads an EPUB from S3.

Example:

```http
GET /api/books/download/Dune.epub
```

---

### Get Cover

```http
GET /api/covers/:filename
```

Returns a cover image from S3.

Example:

```http
GET /api/covers/Dune.jpg
```

---

### Home Page Books

```http
GET /api/books/home/:count
```

Returns a randomized selection of books.

Example:

```http
GET /api/books/home/50
```

Response:

```json
[
  {
    "title": "Dune",
    "filename": "Dune.epub",
    "cover": "/api/covers/Dune.jpg"
  }
]
```

## Environment Variables

Create a `.env` file inside the backend directory.

```env
AWS_REGION=ap-southeast-2

AWS_ACCESS_KEY_ID=YOUR_ACCESS_KEY

AWS_SECRET_ACCESS_KEY=YOUR_SECRET_KEY

S3_BUCKET=YOUR_BUCKET_NAME

PORT=5000
```

Never commit `.env` files to source control.

## Lambda Metadata Synchronization (Planned)

Future workflow:

```text
Upload Cover
        ↓
Upload EPUB
        ↓
S3 Event Trigger
        ↓
Lambda
        ↓
Check Cover Exists
        ↓
Insert Metadata Into RDS
```

The Lambda function will:

1. Detect new EPUB uploads
2. Extract title from filename
3. Check for matching cover
4. Insert metadata into RDS
